## Default Permission

Default permissions for the age-signals plugin — allows checking age range.

#### This default permission set includes the following:

- `allow-check-age-range`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`age-signals:allow-check-age-range`

</td>
<td>

</td>
</tr>

<tr>
<td>

</td>
