## Default Permission

Default permissions for the age-signals plugin — allows checking the age signal.

#### This default permission set includes the following:

- `allow-age-signal`

## Permission Table

<table>
<tr>
<th>Identifier</th>
<th>Description</th>
</tr>


<tr>
<td>

`age-signals:allow-age-signal`

</td>

</td>
</tr>

<tr>
